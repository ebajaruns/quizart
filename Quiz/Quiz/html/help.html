About

This application can be used to create nice quizes! Process of quiz creation is very simple, firstly in the main menu choose "Create", input your Quiz Tilte and small description (later to recognize it). Add questions - each question can have a description or and an image. Input multiple answers, each answer can have picture as well. Add additional questions if neccessary. Save your Quiz on your OneDrive and share it with friends via email. This application was created by teacher and mostly is aimed for teachers and students. App is simple and can provide some fun time!
Enjoy while it lasts!
